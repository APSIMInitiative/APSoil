To release Apsoil:

1. Update Release Notes.htm
2. Recompile the release - ApsoilSetup.sln
3. Run Create Release.bat
4. Copy ApsoilSetup.exe to FTP://www.apsim.info//ProductDownload/Files/APSoil
5. Update the web site: http://www.apsim.info/Wiki/APSoil.ashx


To upload a new .soils file.

1. Copy APSRU-Australia-Soils.soils to  http://www.apsim.info/Wiki/public/Upload/ApSoil
2. Update the web site: http://www.apsim.info/Wiki/APSoil.ashx
3. Upload APSRU-Australia-Soils.soils to http://www.apsim.info/ApsoilWeb/UploadSoilsFile.aspx
